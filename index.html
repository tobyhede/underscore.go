Helpfully Functional* Go

A useful collection of Go utilities. Designed for programmer happiness. 

import "__"
i := []int{1, 2, 3, 4}
__.Contains(i, 3) //true

s := []string{"a", "b", "c", "d"}
m := __.Map(s, func(s string) string) {
	return s + "!"
})
fmt.Println(m) //["a!", "b!", "c!", "d!"]

Not really very fast. But fast enough.
